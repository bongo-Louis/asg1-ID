# asg1-ID

## what's this?
this is a information and concert booking page for the virtual singer, hatsune miku

## design intent
I wanted to design a simple website for users to quickly order miku's latest ticket event, learn more about miku and her songs, as well as stay up to date with the latest merchandise and events that she has to offer.

## website demographic
this website caters to anyone interested in miku, from newcomers to diehard fans.
